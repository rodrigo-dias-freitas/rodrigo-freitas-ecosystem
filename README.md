---

## 🚀 Overview

This project is a fullstack ecosystem focused on **data-driven decision making in financial markets**, combining:

- 📊 Advanced market structure analysis (SFLIM)
- ⚙️ Performance execution methodology (OPX)
- 🎯 Clean and scalable UI/UX (Angular + Tailwind)
- 🔌 API-driven architecture for market data integration

The goal is to transform raw financial data into **actionable insights and structured execution**.

---

## 🧠 Core Concepts

### SFLIM — *Smart Flow & Liquidity Intelligence Model*

A proprietary framework built on three pillars:

- **Context** → Market environment and macro positioning  
- **Structure** → Price behavior, levels, and liquidity zones  
- **Flow** → Momentum, continuation, and execution signals  

---

### OPX — *Operational Performance Execution*

A discipline focused on:

- Execution consistency  
- Risk control  
- Performance tracking  
- Behavioral optimization  

---

## 🏗️ Architecture

The project follows a modular fullstack structure:

/backend → API layer (data processing & endpoints)  
/frontend → Angular dashboard (UI + visualization)  
/public → Static assets / landing  

### Tech Stack

**Frontend**
- Angular (CLI 21+)
- Tailwind CSS
- TypeScript

**Backend**
- Node.js
- REST API architecture

**Others**
- Git-based workflow
- Modular structure (evolving to monorepo)

## ⚙️ Getting Started

### 1. Clone the repository

git clone https://github.com/rodrigo-dias-freitas/rodrigo-freitas-ecosystem.git  
cd rodrigo-freitas-ecosystem  

---

### 2. Install dependencies

npm install  

---

### 3. Run Frontend (Angular)

cd frontend  
npm install  
ng serve  

Access:  
http://localhost:4200  

---

### 4. Run Backend

cd backend  
npm install  
npm run dev
